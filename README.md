**Easy_Learn** یک پروژه‌ی نمونه برای نمایش مهارت‌های توسعه با **ASP.NET Core** است. این پروژه با معماری سه لایه طراحی شده و شامل بخش‌های زیر می‌باشد:

- **Easy_Learn.Data**: لایه دسترسی به داده‌ها و منطق تجاری
- **Easy_Learn.UI**: لایه نمایش و رابط کاربری
- **Easy_learn.WebApi**: لایه سرویس API برای ارتباط با سرویس‌گیرنده‌ها

## 🚀 راه‌اندازی و اجرا

<pre style="font-family: monospace; background: #0D1117; color: #00FF88; padding: 12px; border-radius: 8px; border: 1px solid #30363d;">
1. مخزن را Clone کنید:
   git clone https://github.com/abolfazlmo006/Easy_Learn.git

2. وارد پوشه پروژه شوید:
   cd Easy_Learn

3. بازیابی بسته‌های NuGet:
   dotnet restore

4. پروژه WebApi را اجرا کنید:
   cd Easy_learn.WebApi
   dotnet run

5. پروژه UI را اجرا کنید:
   cd ../Easy_Learn.UI
   dotnet run
</pre>

> **نکته:** برای مشاهده مستندات API پس از اجرای پروژه WebApi، به آدرس `https://localhost:5001/swagger` مراجعه کنید.
